# RukingFrame
这是一个自定义框架

## Step 1. Add the JitPack repository to your build file
	
```
  allprojects {
		repositories {
			...
			maven { url 'https://jitpack.io' }
		}
	}
```
  
## Step 2. Add the dependency
### 自适应框架
``` 
     compile 'com.github.RukingCheng.RukingFrame:autolayout:1.0.0.7'
 ```
     代码来源：[https://github.com/supengchao/AutoLayout](https://github.com/supengchao/AutoLayout)
   
### RK框架
``` 
     compile 'com.github.RukingCheng.RukingFrame:framelibrary:1.0.0.7'
```
  
### 二维码扫一扫
``` 
     compile 'com.github.RukingCheng.RukingFrame:simplezxing:1.0.0.7'
```
     代码来源：[https://github.com/GuoJinyu/SimpleZXing](https://github.com/GuoJinyu/SimpleZXing)
